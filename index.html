An Investigation of False Information and its Detection

Abstract

This thesis investigates aspects of false information and its detection. Artificial intelligence (AI) is currently being used for various tasks, such as social media recommendation, object detection, and speech recognition, etc. In addition, it is creating problems that involve personal and false information. These problems include swaying opinions, creating rumours, scaremongering, and distrusting the government. The methods to combat false information include fact-checking sites (which involve manual processes), automated fact-checking, and user awareness. Large Language Models (LLMs) are neural networks with over a million parameters that are pre-trained on large text datasets and fine-tuned on specific tasks such as question answering, language translation and sentiment analysis. LLMs are useful because they do not require large datasets to be fine-tuned to classify and generate to specific tasks. We make two kinds of contributions. On the technical level, we improved the accuracy and generality of false information detection on a twitter dataset by using LLMs. We also explored explainability of a detection algorithm using LIME and SHAP framework. On the knowledge level, we further investigated the societal aspects of false information issues. First is the social media algorithms which can help to spread false information. The second is the laws and regulations that can help mitigate the issues.

Follow Live

Follow live was made because I was unable to find easy way to find vtubers (Anime Style Avatars) who were live
I used my holiday in 2020 to create a backend and frontend from my previous experiences with ReactJS and NodeJS

One main feature I really like is that it does not rely on Youtube API to find livestreams. I learned a lot about CORS and how to bypass it or to figure out a different way to achieve the same goal. FollowLive uses React 17 and Material UI 4. Some futures includes: notification on your favourite channel, see new channels and add your own channels
